# Interactive Playground 2026

This is a p5.js project to create interactive virtual environments
where users interact via the camera.

## Requirements

- Simple p5.js project suitable for high school students
- Uses OpenCV for computer vision
- There are multiple scenes using a Scene class that handles display and
  any user interaction.  
- The code for each Scene is in a separate .js file
- The user can switch Scenes with a key press. 
- The first scene is a simple example of face detection, drawing an
  outline of the user's face on top of the video.
- The second scene is a simple example of body detection, with a blue
  ball drawn at the center of mass of the user's body.
